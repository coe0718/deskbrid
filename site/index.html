🖥️ AI-powered desktop control

Control your
Linux Desktop

Deskbrid is a lightweight daemon that bridges AI assistants to your Linux desktop. Control windows, inject keystrokes, take screenshots, and manage clipboards — all over a simple Unix socket protocol.

⬇️ Download v0.6.0 View on GitHub →
deskbrid — interactive demo
# Connect to the daemon and control your desktop

$ echo '{"type":"windows.list","id":"req-1"}' | socat - ~/.deskbrid/socket
← 4 windows · active: 0x3a0000b (Firefox)

$ echo '{"type":"windows.focus","id":"req-2","window_id":"0x3a0000b"}' | socat - ~/.deskbrid/socket
← ok · focused: 0x3a0000b

$ echo '{"type":"input.type","id":"req-3","text":"git push origin main"}' | socat - ~/.deskbrid/socket
← typed 20 characters

$ echo '{"type":"screenshot","id":"req-4"}' | socat - ~/.deskbrid/socket
← captured 1920×1080 screenshot

Every control surface, one socket

🪟

Window Control

List, focus, close, minimize, maximize, move, resize — full programmatic window management across any desktop environment.

⌨️

Input Injection

Type text, press keys, click coordinates, drag, scroll. Full keyboard and mouse automation through ydotoold or XTest.

📷

Screenshots

Capture full-screen or region screenshots with real pixel-dimension reporting. GNOME screencast, grim, spectacle, import — whichever fits your DE.

📋

Clipboard

Read and write clipboard contents with explicit format negotiation. Primary and clipboard selections supported on Wayland.

🔔

Notifications

Send desktop notifications with custom urgency, summary, and body. Works via notify-send or native DBus on KDE/GNOME.

📺

Workspaces

List and switch workspaces, move windows between them. Full multi-monitor awareness with per-monitor workspace isolation.

Backend Support

Auto-detected at startup — no config needed

Get started in 30 seconds

# One-shot install — auto-detects desktop + installs deps recommended
$ bash <(curl -fsSL https://deskbrid.patchhive.dev/install.sh)
# Download binary directly
$ curl -LO https://github.com/coe0718/deskbrid/releases/download/v0.6.0/deskbrid && chmod +x deskbrid
# Or build from source
$ git clone https://github.com/coe0718/deskbrid.git && cd deskbrid && cargo build --release
# Start the daemon
$ deskbrid daemon
