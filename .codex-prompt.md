## What to build

### 1. Python client library (clients/python/)
Create a pip-installable Python package at `clients/python/deskbrid/` that wraps the JSON socket protocol. Structure:

```
clients/python/
  pyproject.toml
  deskbrid/
    __init__.py          # Public API
    client.py            # Socket connection + JSON message handling
    events.py            # Event loop + subscription management
    models.py            # Dataclasses for events and action params
```

The client API should be:
```python
from deskbrid import Deskbrid

client = Deskbrid()  # auto-connects to $XDG_RUNTIME_DIR/deskbrid/socket

# Subscriptions
@client.on("window:focus")
def on_focus(data):
    print(data.title, data.app_id)

# Actions
client.type_text("hello world\n")
client.send_keys(["ctrl", "shift", "t"])
client.mouse_click(x=100, y=200, button="left")
text = client.clipboard_read()
client.clipboard_write("copied text")
path = client.screenshot()
client.notify("Title", "Body text")
windows = client.list_windows()
client.focus_window(app_id="firefox")
monitors = client.list_displays()
info = client.info()

# Start listening for events (runs until Ctrl+C)
client.listen()
```

Features:
- Uses Unix socket (connect to $XDG_RUNTIME_DIR/deskbrid/socket)
- Async OR sync — support both with a sync convenience wrapper
- JSON-line protocol matching PROTOCOL.md exactly
- Dataclasses for structured responses (WindowInfo, MonitorInfo, ClipboardContent)
- Error handling — raise DeskbridError for non-ok results
- Auto-reconnect on connection drop

### 2. Rust CLI tool (src/cli.rs)
Add a built-in CLI mode to the deskbrid crate. When run as `deskbrid-cli subscribe window:focus clipboard`, it connects to a running daemon, subscribes, and prints events to stdout.

CLI commands:
- `deskbrid daemon` — runs the daemon (default, current behavior)
- `deskbrid subscribe [events...]` — subscribe and stream events as JSON lines to stdout
- `deskbrid action <action> [params_json]` — send a single action and print the result
- `deskbrid info` — print daemon info
